PHP提供了很多很好用的内置函数，用好这些函数可以提高自己的效率，所以没事应该多看看php手册。

int substr_count ( string haystack, string needle)

函数可以统计haystack字符串中needle字符串的出现频率。

PS:

haystack 草垛

needle 针
