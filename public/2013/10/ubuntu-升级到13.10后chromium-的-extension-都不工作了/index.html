Canonical 前段时间刚发布了 Ubuntu 13.10 (Saucy) , 作为Ubuntu的死忠赶紧升级到最新版本，但是悲剧接着就发生了，Chromium  的所有扩展(extension)都不能用了，找了大半天也没有找到很好的解决方案。

最后只好换了Google Chrome，搞定之后发现还是chrome要好用些。

通过下面的命令就可以安装：

// add key
wget -q -O - https://dl-ssl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
// add it to the repository
sudo sh -c 'echo "deb http://dl.google.com/linux/chrome/deb/ stable main" >> /etc/apt/sources.list.d/google-chrome.list'
// update
sudo apt-get update
// install google chrome
sudo apt-get install google-chrome-stable

也可以直接去chrome官网下载deb包，直接安装就好了。

PS：Chrome 和 Chromium 的区别
