在CentOS继续折腾MySQL的编译安装······，安装过程中常遇到gcc编译器缺失和No curses/termcap library found问题。解决方案总结如下

1）gcc编译器缺失

运行命令：yum install gcc-c++

2）No curses/termcap library found

通过命令安装** **curses/termcap库 yum install ncurses-devel
